Hi, I'm Harry

Bio

I'm a London based software developer with years of experience writing open source cross-platform software in C, C++, Python, and Go.

I'm primarily a back-end developer who gets his kicks from optimising slow software and designing service architectures.

My design philosophy borrows from the UNIX way: Write clean software that does one thing, and does it well. Strive to write clean, beautiful code, but where performance demands, optimise aggressively.
